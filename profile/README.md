# 🎨 MAC (Make A Chart)

> 비개발자를 위한 한국형 No-Code 시각화 웹서비스

---

## 🧭 Overview

MAC은 데이터를 넣기만 하면
누구나 빠르게 차트를 생성하고 수정할 수 있는 **시각화 편집 도구**입니다.

기존 복잡한 분석 툴과 달리,
**단계형 사용자 흐름**을 통해 비전공자도 쉽게 결과물을 만들 수 있도록 설계되었습니다.

---

## 🎯 Core Message

> **MAC은 전문가용 분석 툴이 아니다.**

MAC은
👉 데이터를 빠르게 시각화하고
👉 원하는 스타일로 쉽게 수정할 수 있도록 돕는

**한국형 No-Code 시각화 편집기**를 목표로 합니다.

---

## 🚀 Goal

* 📅 **2026년 6월 15일 베타 출시**
* 차트 생성 → 편집 → 저장 → 다운로드 → 공유까지 가능한 서비스 완성

---

## ⚠️ Problem

기존 시각화 도구들은

* 영어 중심 UI
* 복잡한 설정 구조
* 과도한 선택지 제공

👉 **비전공자가 사용하기 어려움**

---

## ✅ Solution

MAC은 아래와 같은 흐름으로 문제를 해결합니다:

```
차트 선택 → 데이터 입력 → 옵션 조정 → 결과 확인 → 저장/공유
```

👉 사용자는 도구를 배우는 것이 아니라
👉 **단계를 따라 결과물을 만드는 경험**을 하게 됩니다.

## 🧩 Template-Based Architecture

MAC은 차트를 “템플릿 구조”로 관리합니다.

### 📌 예시

| 차트           | 데이터 구조           | 주요 옵션          |
| -------------- | --------------------- | ------------------ |
| Bar Chart      | category, value       | 색상 / 정렬 / 라벨 |
| Line Chart     | x, value              | 선 두께 / 마커     |
| Pie / Donut    | category, value       | 색상 / 도넛 두께   |
| Bar Chart Race | time, category, value | 속도 / top N       |

---

### 💡 장점

* 차트 확장 용이
* 유지보수 효율성
* FE/BE 협업 구조 통일
